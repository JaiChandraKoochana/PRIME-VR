# PRIME VR

> A Virtual Reality sports training platform that combines motion tracking, player analytics, and immersive VR experiences to help athletes improve their performance.

## 📖 Overview

PRIME VR is a web-based application designed to provide an interactive Virtual Reality experience for sports training. The platform enables users to create accounts, manage profiles, upload content, track performance statistics, and explore VR-powered training modules.

The project focuses on creating an engaging environment where players can analyze their movements and improve their skills using immersive technologies.

## 🚀 Getting Started

### Prerequisites

* Node.js (v18 or later recommended)
* npm

### Installation

```bash
git clone https://github.com/your-username/PRIME-VR.git

cd PRIME-VR

npm install

npm run dev
```

The application will start on your local development server.
